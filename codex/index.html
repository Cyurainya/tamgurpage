三步完成配置

每一步一张大图，照着截图操作即可。

模型中心编辑配置应用管理
1

添加模型

进入 模型中心，点击“添加模型”，选择 OpenAI / Anthropic API 类型。

截图 1：在模型中心点击添加模型。
2

填写 API 配置

OpenAI 地址填中转站接口并带 /v1；模型 ID 填模型名；API 密钥填令牌 Key。

截图 2：填写地址、模型 ID 和 API Key。
3

绑定 Codex

进入 应用管理，选择 Codex Desktop，绑定刚添加的模型，然后点击“启动应用”。

截图 3：为 Codex Desktop 选择模型并启动。

配置要点

填错最多的地方，按这里核对。

填写规则

  • 名称只是显示名，可以写成 GPT5.5、KK-GPT、Codex 中转站等。
  • OpenAI 地址填写 https://api.tamgur.tech/v1。
  • 模型 ID 要和中转站后台支持的模型名称一致，建议直接复制。
  • API 密钥填中转站令牌 Key，如果有分组，记得选对分组权限。

配置模板

把下面内容替换成你自己的中转站信息即可。

名称：GPT5.5
OpenAI 地址：https://api.tamgur.tech/v1
Anthropic 地址：不用可留空
模型 ID：请从模型广场复制
API 密钥：sk-xxxxxxxxxxxxxxxx
